## 🔄 Automation

This repository is automatically updated daily via GitHub Actions!

- **Schedule**: Daily at 9:00 AM UTC
- **Data Source**: Open-Meteo API (Free, no API key)
- **Coverage**: 100+ cities across 40+ countries

### How It Works

1. GitHub Actions triggers daily at 9 AM UTC
2. Python script fetches weather from Open-Meteo API
3. Data is processed and saved to JSON/CSV files
4. README is auto-generated with statistics
5. Changes are auto-committed to repository
